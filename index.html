How to apply CSS to HTML

To apply an external CSS file within a HTML file you need to use the link tag inside the head of your HTML document.

Example: <link rel="stylesheet" href="styles.css">

In the example above, rel="stylesheet" tells your browser that the file is a stylesheet, while href="styles.css" will specify the path to your CSS file. If the CSS file is in the same directory as your HTML file, the file name should suffice. Otherwise, you must provide the full path to the CSS file.

How to use JavaScript in a Webpage

To link an external JavaScript file within a HTML file you need to use the script tag, either in the head or just before the closing body tag within the HTML document.

Example: <script src="script.js"></script>

In the example above src="script.js" specifies the location of the JavaScript file. If the file is in the same directory as the HTML file, you only need the file name. Otherwise, you must provide the full path to the file.

To write JavaScript directly within a HTML file, you still need to use the script tag. Though it is possible to write it within the head of the HTMl file, it is best to write it before the end of the body section to make sure page content is loaded before the script runs.

Example:

<body>

 <script>

   //Javascript code goes here

   console.log("Javascript at bottom of body is running");

 </script>
